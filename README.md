# Python Simple Calculator

A simple calculator built using Python.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by zero handling

## Concepts Used

* Variables
* User Input
* If Statements
* Arithmetic Operators

## How to Run

1. Run the Python file.
2. Enter the first number.
3. Choose an operation (+, -, *, /).
4. Enter the second number.
5. View the result.
